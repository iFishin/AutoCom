# 用户自定义场景示例（预留）

把你常用的业务场景放到这里，autocom-helper 会优先参考此文件来生成配置。

## 示例模板

### 场景名: WiFi 重连稳定性

- 目标: 连续重连 100 次，统计失败率
- 设备: DUT_WiFi
- 关键指令:
  - AT+CWMODE=1
  - AT+CWJAP="...","..."
  - AT+CWQAP
- 推荐超时:
  - 连接: 15000
  - 断开: 5000
- 备注:

### 场景名: BLE 广播扫描回归

- 目标: 广播开启后，扫描结果中必须包含目标名称
- 设备: DUT_BLE
- 关键指令:
  - AT+BLENAME="..."
  - AT+BLEADV=1
  - AT+BLESCAN=1,"10"
- 推荐超时:
  - 扫描: 15000
- 备注:
